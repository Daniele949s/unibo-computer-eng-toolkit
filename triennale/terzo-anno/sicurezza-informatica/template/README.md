# Librerie per il corso Sicurezza Informatica T
## Autore: Davide Chirichella (https://github.com/chirichexe)
## Anno accademico 2024-2025

La struttura della libreria è la seguente:

```
    - utils
	utilities generali, che possono servire in tutti gli esercizi 
    
    - [esercizio]
	- utils
	    utilities per l'esercizio nello specifico
        - report.txt
	    un template per il report o per qualsiasi altro
	    file da consegnare
	
	    ! IMPORTANTE ! Cambiare la matricola nell'intestazione dei file

    - Makefile
	mediante il comando "make consegna < es1 ... esn>" è possibile
	creare l'archivio .zip da consegnare. Eventuali cartelle "utils"
	all'interno degli esercizi verranno ignorate
	
	! IMPORTANTE ! Cambiare la matricola

	mediante "make clear" si elimina un'eventuale zip della consegna
```
