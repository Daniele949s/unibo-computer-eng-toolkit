# Librerie per il corso Sistemi Informativi T
## Autori: Davide Chirichella (https://github.com/chirichexe) e Filippo Giulietti (https://github.com/FiloGiulietts)
## Anno accademico 2023-2024

Raccoglie una serie di librerie SQL utili per l'esame di Sistemi Informativi, tenuto dal professor Paolo Ciaccia presso l'Università di Bologna. 

Le librerie sono organizzate per coprire i principali argomenti del corso, tra cui la creazione di database e tabelle, l'inserimento e manipolazione dei dati, l'uso di trigger, viste e subquery, oltre alle operazioni di aggregazione e gestione dei vincoli di integrità. 

Ogni sezione include esempi pratici per facilitare la comprensione e l'applicazione dei concetti.

Il formato è il `Markdown`, che è leggibile più chiaramente aprendolo con VScode -> `CTRL + shift + V`.

Non mi assumo nesssuna responsabilità per eventuali bocciature :)

Buono studio!
